# Todo App

Minimal JS todo app with add/edit/filter, drag-and-drop reordering, light/dark theme toggle, animations, and local storage persistence.

## Features
- Add, edit, delete tasks
- Filter: All / Active / Completed
- Clear completed
- Drag & drop reordering
- Light/dark theme toggle (persists)
- Keyboard-friendly input (Enter to add)
- Local storage persistence

## Stack
- HTML, CSS, JavaScript (no frameworks)
- LocalStorage for persistence

## Quick Start
1. Clone: `git clone https://github.com/<you>/todo-app.git`
2. Open `todo-app/index.html` in a browser (no build needed).

## Notes
- Drag & drop works within the filtered view.
- Theme preference is saved in localStorage.
